# Findar App Icon

## Add the app icon here:

Save the blue Findar icon as:
- **Filename:** `findar-icon.png`
- **Location:** This folder (`public/images/`)

The icon will automatically appear in:
- ✅ Navbar (top left with "Findar" text)
- ✅ Footer (bottom with company info)

## Icon Specs:
- Format: PNG (with transparency preferred)
- Size: 512x512px or larger
- The current blue icon with house and magnifying glass design
